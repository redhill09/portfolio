About

Getting to know me

Trust the process
nothing is easy but it's worth it.

I started as a graphics designer knowing just one simple application, Adobe Illustrator; and then I realised that technology never stops evolving, there’s always something new everyday, and if I don’t step up my game I will not be able to keep up. So I started learning the basics on creating a marketing materials then animations using different softwares, finding what’s the best for each task, after some time, I’m creating website layouts and mobile application layouts; and now I’m a Front End developer. I never imagine that I can do these kind of things and I enjoy being a Frond End developer. Creating a clean, elegant, user-friendly and of course responsive layouts is one of my passions today.

Website Development

Mobile Development

Graphics Design

Create marketing materials using Adobe Photoshop, Adobe Illustrator and Adobe Lightroom

Animation & Video Editing

Create simple animation and edit videos using Adobe Flash and Adobe Premiere
